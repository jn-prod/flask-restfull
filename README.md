Flask-RESTful provides the building blocks for creating a great REST API.

## Installation
```
pip install virtualenv
virtualenv env -p python3
. env/bin/activate
pip install flask
```

https://flask.palletsprojects.com/en/1.1.x/

## init

```
python run.py
```

## launch server

```
python3 server.py
```
